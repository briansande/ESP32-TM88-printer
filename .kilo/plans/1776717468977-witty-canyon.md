# Plan: Add thermalprinter.cpp Library Functions with Web UI

## Overview
Port all functions from [signalwerk/thermalprinter](https://github.com/signalwerk/thermalprinter) into the existing `main.cpp`, adapting from the library's class-based `SoftwareSerial` pattern to flat static functions using the existing `HardwareSerial printer(2)`.

## New HTTP Endpoints

| Endpoint | Method | Params | Calls |
|---|---|---|---|
| `/status` | GET | — | `printerGetStatus()` → returns JSON `{status, raw}` |
| `/bold` | POST | `on=1\|0` | `printerBoldOn/Off()` |
| `/underline` | POST | `on=1\|0` | `printerUnderlineOn/Off()` |
| `/doubleHeight` | POST | `on=1\|0` | `printerDoubleHeightOn/Off()` |
| `/reverse` | POST | `on=1\|0` | `printerReverseOn/Off()` |
| `/justify` | POST | `pos=left\|center\|right` | `printerJustifyLeft/Center/Right()` |
| `/lineSpacing` | POST | `n=0-255` | `printerLineSpacing(n)` |
| `/defaultLineSpacing` | POST | — | `printerDefaultLineSpacing()` |
| `/characterSet` | POST | `n=0-10` | `printerCharacterSet(n)` |
| `/barcode` | POST | `m=65-73, data=string` | `printerPrintBarcode(m, data)` |

## Web UI Updates

Add a second section to the dashboard card with:
- **Style toggles row**: Bold, Underline, Double-Height, Reverse — toggle buttons (on/off styling)
- **Justify row**: Left / Center / Right — radio-style buttons
- **Line Spacing**: number input (0-255) + "Set" button + "Default" button
- **Character Set**: dropdown (USA, France, Germany, UK, etc.)
- **Barcode section**: text input for data, dropdown for code type (65-73), "Print Barcode" button
- **Status**: button to fetch/refresh status

All new UI follows the existing dark theme (`#1a1a2e` bg, `#e94560` accent, `#0f3460` buttons).
